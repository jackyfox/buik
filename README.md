# БУИК

Исходники прошивки для блока управления импульсным клапаном (БУИК) для системы газовой аварийной защиты. Девайс контролирует состояние быстродействующего отсечного газового клапана и умеет закрывать его по внешнему сигналу от газосигнализатора или при обрыве питания

Архитектура:  **AVR**

Чип: **ATTINY13A**

Код: **C**
